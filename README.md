# osu-bot

Quickstart
--------

```bash
git clone https://github.com/fumycat/osu-bot.git
cd osu-bot/
python3 osu.py
```

Requirements
--------

- [Requests](https://github.com/kennethreitz/requests/)
- [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)

Osu! api access - https://osu.ppy.sh/p/api
